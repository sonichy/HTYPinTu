# Qt 海天鹰拼图
基于 Qt 的拼图程序，横向或竖向排列，保存拼图。  
已编译好的 HTYPinTu 程序适用 64 位 Linux 系统 Qt5 环境，双击运行，其他版本自行编译。  

![alt](preview.png)
